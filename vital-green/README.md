# Vital Green Frontend

React + Vite frontend for the Vital Green project.

## Quick Start

```bash
cd vital-green
npm install
npm run dev
```

Frontend runs on `http://localhost:5173` by default.

## Environment Variables

- `VITE_API_URL` - Base URL for the backend API (for example: `http://localhost:5000/api`)
- `VITE_MAIN_SITE_URL` - Canonical site URL used for redirects/links
